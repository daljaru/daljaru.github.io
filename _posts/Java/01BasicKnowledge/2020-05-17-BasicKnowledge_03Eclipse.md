### IDE를 설치하자. 

IDE(Integrated Development Environment)는 개발자가 소스코드 쉽게 작성할 수 있도록 도와주는 프로그램입니다. 그리고 컴파일과 실행을 자동으로 해주고 디버깅도 지원합니다. 

개발자는 원하는 코드를 작성하고 프로그램 실행결과를 바로 확인 할 수 있습니다. 

Java 파일을 만들기 위해 Eclipse IDE를 설치할 것입니다.
